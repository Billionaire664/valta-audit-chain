# Contributing

This is a small reference implementation, kept intentionally minimal. Contributions welcome:

- Bug fixes to the hash chain / spend gate logic
- Ports to other languages (Python, Go, Rust — happy to link them from the README)
- Additional real-world edge cases in the "design notes" section

Before opening a PR: run `npm run typecheck && npm run demo` and make sure both pass.

For anything larger than a small fix, open an issue first to discuss — keeps the repo focused.
